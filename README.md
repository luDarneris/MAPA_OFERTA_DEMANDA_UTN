1. Requisitos previos:
Tener instalado Git, Docker Desktop y Power BI Desktop
3. Clonar el repositorio: 
git clone https://github.com/luDarneris/MAPA_OFERTA_DEMANDA_UTN.git
4. Levantar los contenedores:
    - Abrir Docker Desktop
    - Abrir la consola de git en la carpeta principal del repositorio clonado y  ejecutar docker compose up -d
5. Credenciales para conexión DW: datos necesarios en el archivo .yml
6. Para visualizar los tableros de comando en Power BI: Se debe activar la visualización de mapas 
    - File -> Options and Settings -> Global -> Security -> Map and Filled Map Visuals (Activar opción) -> Ok -> Reiniciar Power BI
